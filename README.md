# Tennis World

A modern, interactive Duolingo-inspired tennis training web app built with HTML, CSS, and JavaScript, designed to teach players tennis techniques, track progress, test knowledge, and motivate improvement through gamification, achievements, levels, and a smart coaching system.


## Features

- User Accounts (Login & Register)
  - Each user has their own:
    - Progress
    - Favorites
    - Quiz history
    - Achievements
    - XP, level, and streak

- Welcome Page
  - Modern split hero layout with images
  - Learning paths (Beginner, Intermediate, Advanced)
  - Live progress summary
  - Smart Coach recommendations

- Shots Learning System
  - Learn shots with:
    - Technique
    - Common mistakes
    - Drills
    - Coaching tips
  - Mark shots as:
    -  Learned
    -  Favorite
  - Filter by:
    - All
    - Favorites
    - Learned

- Progress Tracking
  - Tracks:
    - Learned shots
    - Completion percentage
    - Favorites
    - XP and level
    - Daily streak

- Quiz System
  - Randomized questions
  - Instant feedback:
    - Shows correct / wrong answer
  - Final score summary
  - Restart quiz option
  - Quiz history saved per user

- Achievements System
  - Unlock achievements for:
    - Learning shots
    - Completing quizzes
    - High scores
    - Streaks
    - Level-ups
  - Big animated popup when unlocked
  - Achievements shown in Profile

- XP & Level System
  - Gain XP by:
    - Learning shots
    - Completing quizzes
  - Automatic leveling system
  - Level-up animation popup

- Daily Streak System
  - Tracks:
    - Current streak
    - Best streak
  - Updates automatically each day you open the app

- Smart Coach (AI-like System)
  - Analyzes:
    - Unlearned shots
    - Favorites
    - Quiz mistakes
    - Recent performance
  - Recommends top 3 focus areas
  - Uses priority levels:
    - High / Medium / Low

- Skill Roadmap
  - Visual learning progression
  - Shows what to learn next
  - Shows locked & unlocked skills

-  Profile Dashboard
  - Shows:
    - Username
    - Progress stats
    - XP & Level
    - Streaks
    - Achievements
    - Quiz history
    - Charts

-  Smart Search
  - Search across:
    - Shots
    - Sections
    - Pages
